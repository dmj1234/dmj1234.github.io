CSS居中div的几种方法

发表于 2020-12-16 更新于 2020-12-12

第一种：text-align:center

.center{
　　text-align:center;

}
center_text{
　　display:inline-block;
　　width:500px
}

这种方式可以水平居中块级元素中的行内元素，如inline，inline-block；

  • 但是如果用来居中块级元素中的块级元素，如div中的div，一旦内层的div有自己的宽度，这种方法就会失败。只能让里面的div的文字等内容居中，而div仍然是左对齐的。
  • 还有一种情况，当内部的元素脱离了文档流，前提display:absolute的情况下，不管是否是块级元素，都会居中，但是这种居中不是基于内部div的内容，而是内部div最左端，内部div的最左端在div的中间。

第二种：margin:0 auto

center_text{
　　display:block;
　　width:500px
　　margin：0 auto；
}

这种方式要求内部元素是块级元素，并且不能脱离文档流（如设置position：absolute，否则无效。

第三种方式：脱离文档流的居中方式

.mask{
　　display: block;
　　position: fixed;
　　top: 0;
　　left: 0;
　　width: 100%;
　　height: 100%;
　　background: #000;
　　z-index: 10000;
}
.center{
display: block;
position: fixed;
position: absolute;
top: 50%;
left: 50%;
width: 666px;
height:400px;
margin-left: -333px;
margin-top: -200px;
z-index: 10001;
box-shadow: 2px 2px 4px #A0A0A0, -2px -2px 4px #A0A0A0;
background-color: #fff;
}

  • 这种居中方式，把内部div设置宽度和高度后，再设置top、left各位50%，设置完成之后（这里是按照左端居中的）接着我们使用负边距的方式调整，将margin-top设置为负的高度，margin-left设置为负的宽度的一半，就可以居中了。

第四种：使用CSS 3的translate水平垂直居中元素

.center {
position: relative;
height: 500px;}
.center_text{
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
width: 300px;
height: 600px;
}
