发表于 2020-12-20 更新于 2020-12-23
  • 先看代码 函数调用：
  • JS（ES5）里面有三种函数调用形式：
  • 第三种形式，才是正常调用形式
    func.call(context,p1,p2)
  • 其他两种可以转换为call的形式
  • 至此，我们的函数调用形式只有一种：
    func.call(context, p1, p2)
  • this就是上面的context，this是我们call一个函数时传的context。

[ ] 语法

  • 我们可以把 arr0 想象为arr.0( )，虽然后者的语法错了，但是形式与转换代码里的 obj.child.method(p1, p2) 对应上了，于是就可以愉快的转换了：
  •     arr[0]()
    假想为 arr.0()
    然后转换为 arr.0.call(arr)
    那么里面的 this 就是 arr 了 :)

箭头函数

  • 箭头函数里并没有 this，如果你在箭头函数里看到 this，你直接把它当作箭头函数外面的 this 即可。外面的 this 是什么，箭头函数里面的 this 就还是什么，因为箭头函数本身不支持 this。
