Ajax和Json

发表于 2020-12-20

什么是JSON

  • JavaScript Object Notation，在JS语言中，一切都是对象。任何JS支持的都可以通过JSON来表示，例如字符串，数字，对象，数组等。

  • JSON键值对是用来保存JS对象的一种方式，和JS对象的写法大同小异，键/值对组合中的键名写在前面并用双引号””包裹，使用冒号分隔，然后紧着值。

JSON和JS对象的转换

  • 要实现从JSON字符串转换为JS对象，使用JSON.parse()方法：
    var obj = JSON.parse(‘{“a” : “”Hello}’);
  • 要实现JS对象转换为JSON字符串，使用JSON.stringfy()方法：
    var json = JSON.stringfy({a: ‘Hell0’});

什么是虚拟DOM

  • 虚拟DOM的本质就是一个JS对象，虚拟DOM减少了真实DOM的操作，当修改数据的时候，就是修改虚拟DOM产生全新的虚拟DOM，
  • 新旧虚拟DOM使用diff算法，得到patch(也就是需要修改的部分)，然后将这个patch打到浏览器的DOM上（减少重绘和回流，从而达到性能优化的目的）
